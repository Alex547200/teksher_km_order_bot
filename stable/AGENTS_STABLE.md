# Teksher KM Order Bot — правила работы Codex

## Главное правило
Не создавать новый workflow, если задача — исправить существующий скрипт.

## Основной рабочий файл
download-all-2026-05-16.js

## Запрещено без отдельного разрешения
- переписывать весь скрипт;
- создавать новый скрипт вместо исправления старого;
- менять manual-token.js;
- менять endpoint CSV;
- использовать GTIN из DataMatrix-кода для имени файла;
- смешивать операции 15 и 16 мая;
- удалять рабочие файлы.
