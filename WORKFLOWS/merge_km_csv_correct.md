# Merge KM CSV Correct

Назначение:

- объединить ZIP или папку с CSV в один корректный CSV;
- сохранить один КМ целиком в одной строке;
- проверить количество КМ, дубли, длину и GS-разделители.

Требования:

- использовать `csv.reader` и `csv.writer`;
- не использовать `pandas` для КМ;
- не разрывать одну строку DataMatrix/КМ на несколько строк;
- валидировать количество строк, дубли и длину полей;
- проверять наличие ASCII GS `0x1D` и не терять его.

Итоговые файлы:

- `*_KM_CORRECT.csv`
- `*_KM_CHECK.txt`

Что нельзя:

- не использовать `pandas` для объединения КМ;
- не заменять символы GS пробелами;
- не резать КМ на части по столбцам произвольно.
