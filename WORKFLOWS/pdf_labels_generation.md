# PDF Labels Generation

Назначение:

- пересобирать PDF-этикетки Текшер из CSV/Excel;
- сохранять DataMatrix и товарный текст на одной странице 58×40 мм;
- использовать существующие товарные данные, а не захардкоженные шаблоны.

Правила:

- один КМ = одна PDF-страница;
- DataMatrix рисуется настоящим `zint`;
- текст слева должен соответствовать mapping по GTIN/товару;
- размер страницы 58×40 мм;
- не использовать один и тот же товарный блок для всех GTIN;
- при отсутствии mapping остановиться, а не генерировать неверные PDF.

Результаты:

- PDF-папка на Desktop;
- zip-архив с PDF;
- mapping-файл `gtin_mapping.csv`, если используется исходный справочник.

Что нельзя:

- не печатать с заглушками;
- не подменять товарные данные;
- не использовать старые PDF как источник истины для текста.
